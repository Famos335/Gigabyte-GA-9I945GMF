## О репозитории

Этот репозиторий хранит комплект BIOS **F8** для платы Gigabyte GA-9I945GMF: сам образ BIOS, DOS-утилиту прошивки и исходный самораспаковывающийся архив.

Главная цель репозитория — сохранить файлы в одном месте и сделать их содержимое понятным: что именно лежит внутри, зачем нужен каждый файл и на что обратить внимание перед прошивкой.

## Как использовать

1. Сверьте точную модель и ревизию платы по маркировке на самой материнской плате.
2. Проверьте контрольные суммы файлов из [`CHECKSUMS.txt`](CHECKSUMS.txt).
3. Подготовьте DOS-загрузочный носитель.
4. Скопируйте на него `945gmf.f8` и `FLASH892.EXE`.
5. Прошивайте BIOS только если понимаете процедуру и риски.

## Важное предупреждение

Прошивка BIOS может вывести материнскую плату из строя при ошибке модели, сбое питания, поврежденном файле или неправильной команде прошивки.

Обратите внимание: оригинальный архив в этом репозитории называется `motherboard_bios_ga-8i945gmf_f8.exe`, а репозиторий — `GA-9I945GMF`. Перед использованием обязательно проверьте совместимость файла с вашей конкретной платой и ее ревизией.

## Контрольные суммы

SHA-256 суммы вынесены в [`CHECKSUMS.txt`](CHECKSUMS.txt). Их можно проверить командой:

```powershell
Get-FileHash -Algorithm SHA256 -LiteralPath "Gigabyte GA-9I945GMF\945gmf.f8"
```
